# Nyarla’s Birthday Bash

*Main Story* : Witch Lilith tries to bake a cake for her cat, Nyarlathotep’s birthday, however she is missing some ingredients. Help Lilith find them. 

## File Structure
```
/csci499-capstone
├─ assets/
├─ scenes/
├─ scripts/
├─ project.godot
├─ .godot/
├─ .editorconfig
└─ README.md
```

## How to run locally
1. Ensure **Godot 4.6** is installed.
2. Clone the repository:

   ```bash
   git clone https://github.com/jerichom08/csci499-capstone.git
   cd csci499-capstone
   ```

3. Launch Godot 
4. In the **Project Manager**, click **Import**.
5. Select the `project.godot` file at the repository root.
6. Click **Import & Edit**.
7. Press **Play** or **F5** to run the game.

Alternatively, the project can be launched directly from the command line:

```bash
Godot_v4.6-stable_win64.exe --path .
```

If the project fails to run, verify that the correct Godot version is being used and that all files are up to date.
