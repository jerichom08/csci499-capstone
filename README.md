# Nyarla’s Birthday Bash

*Main Story* : Witch Lilith tries to bake a cake for her cat, Nyarlathotep’s birthday, however she is missing some ingredients. Help Lilith find them. 

## Play the Game (Recommended)

The easiest way to play is through our pre-built release:

1. Go to the **Releases** section of this repository  
2. Download the latest version  
3. Follow the instructions provided in the release  

## How to Run Locally (Development)

1. Ensure **Godot 4.6.1** is installed  
   - Download from: https://godotengine.org/

2. Clone the repository:
   ```bash
   git clone https://github.com/jerichom08/nyarlas-birthday-bash
   cd nyarlas-birthday-bash

3. Launch **Godot**

4. In the **Project Manager**, click **Import**

5. Select the project folder or the `project.godot` file at the repository root and import the project.

6. The editor should automatically open. Press **F5** to run the main scene from within the editor. Otherwise, select **Nyarla’s Birthday Bash** from the project list and click **Run**
